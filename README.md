# TetrArcade

Tetris clone made with Python and Arcade graphic library

## Requirements

* [Python](https://www.python.org/)

## Install

```shell
git clone https://git.malingrey.fr/adrien/TetrArcade.git
cd TetrArcade
python -m pip install -r requirements.txt
```

## Play

```shell
python tetrarcade.py
```
